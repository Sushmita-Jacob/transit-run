# transit-run
This project was made for Hack Club: The Game's TransitTime boost (however I reshipped after the boost). It is highly based off of the popular web game Frogger (the old school version of Crossy Road!), where the player has to avoid getting hit by cars or trains. Control the cat using the up arrow and avoid getting hit by a bus or falling into the water. Press the down key to hide! Win by reaching the other side of the road without getting run over! Get a score of 5 to win! If you collide with an object 10 times, the cat dies and you lose.

The following YouTube tutorial was used for the making of this project:
https://www.youtube.com/watch?v=GXvNEwu9cgM

Past the tutorial, the following was added:
- Custom pixel art for water (the tutorial asset looked a little plain)
- Custom cat character instead of a frog!
- Pixelated background for an old-school arcade feel!
- Conditions to win or lose (instead of an infinite game)
- A "You won!" and "You lost!" screen
- Ability to replay if won
- Ability to press the down arrow to hide and avoid collision
- Background color and instructions to play the game
