# transit-run
This project was made for Hack Club: The Game's TransitTime boost (however I reshipped after the boost). It is highly based off of the popular web game Frogger (the old school version of Crossy Road!), where the player has to avoid getting hit by cars or trains. Control the frog using arrow keys (up, down, left, and right) and avoid getting hit by a bus or falling into the water. Win by reaching the other side of the road without getting run over! Get a score of 5 to win! If you collide with an object 10 times, the frog dies and you lose.

The following YouTube tutorial was used for the making of this project:
https://www.youtube.com/watch?v=GXvNEwu9cgM

Past the tutorial, the following was added:
- Custom pixel art for water (the tutorial asset looked a little plain)
- Conditions to win or lose (instead of an infinite game)
- A "You won!" and "You lost!" screen
- Ability to replay if won
- Background color and instructions to play the game
