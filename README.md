# transit-run
This project was made for Hack Club: The Game's TransitTime boost. It is highly based off of the popular web game Frogger (the old school version of Crossy Road!), where the player has to avoid getting hit by cars or trains. Control the frog using arrow keys (up, down, left, and right) and avoid getting hit by a bus or falling into the water. Win by reaching the other side of the road without getting run over!
